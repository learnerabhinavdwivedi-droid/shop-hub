🛒 Shop Hub
A lightweight shopping hub web application built with HTML, CSS, and JavaScript, designed to demonstrate modular coding, clean UI, and backend integration. This project showcases a simple yet scalable architecture suitable for browser extensions or small-scale e-commerce prototypes.

📂 Project Structure
Code
shop-hub/
│── backend/              # Backend logic and API handling
│── icons/                # App icons and assets
│── content.js            # Core content script
│── manifest.json         # Extension manifest configuration
│── popup.css             # Styles for popup UI
│── popup.html            # Popup interface
│── popup.js              # Popup functionality
│── preview_demo.html     # Demo preview page
│── render.yaml           # Deployment configuration
🚀 Features
Popup UI: Interactive shopping hub accessible via browser extension popup.

Backend Integration: Modular backend folder for API routes and logic.

Custom Styling: Clean and responsive design with popup.css.

Demo Preview: preview_demo.html for quick testing without installation.

Deployment Ready: render.yaml included for cloud deployment.

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: Node.js (modular backend folder)

Deployment: Render (via render.yaml)
